# ☁️ Sky Quest - Number Guessing Adventure 🎯

A fun and interactive **Number Guessing Game** developed using **HTML, CSS, and JavaScript**.  
The project provides an engaging gaming experience with animated visuals, multiple difficulty levels, score tracking, timer challenges, and a responsive sky-themed interface.

## 📌 Project Overview

**Sky Quest** is a browser-based number guessing game where players attempt to find a randomly generated number within a selected difficulty level.

The game provides real-time feedback, tracks attempts, calculates scores, and delivers an interactive gaming experience through animations and dynamic UI elements.

## 🎯 How to Play

1. Select your preferred difficulty level.
2. Enter your guessed number.
3. Click the guess button.
4. Use the hints to find the correct number.
5. Complete the challenge before:
   - Attempts run out
   - Timer reaches zero
